# Moonlight Story House｜月光下的故事館

*An AI-powered bedtime storytelling project combining narration, illustration, and emotional design.*

---

## 1. Project Overview｜專案簡介

### Moonlight Story House is an AI-assisted storytelling project designed to explore how narrative design, voice generation, and visual mood-building can work together to create calm, immersive bedtime stories.

This project includes:

- Gentle narration generated with AI text-to-speech  
- Original illustrations created through AI image models  
- Multi-genre story design (horror, post-apocalyptic, and healing forest tales)  
- YouTube publishing workflow, including subtitles and branding  

The goal is to combine creative writing and technology to craft a soothing night-time listening space.

### 「月光下的故事館」是一個以 AI 輔助創作的故事專案，探索敘事設計、語音合成與視覺氛圍如何結合，打造沉靜、沉浸式的睡前故事。

內容包含：

- 以 AI TTS 生成的溫柔旁白  
- 以 AI 生成的故事插畫  
- 三大故事系列：靈異、末日、森林療癒  
- YouTube 上架流程：字幕、封面、品牌系統  

目標是結合創意寫作與技術，打造一個溫柔的夜間聆聽空間。
